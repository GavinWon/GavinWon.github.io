Race App

A fitness app that helps connect runners from around the world and help runners improve their running time. Runners can enter into a queue system and compete in simulated races with other runners from around the world, or complete time trials to improve running times.

App built with Xcode Swift (CLLocation for location tracking/mapping) and Firebase Database and Storage API.

Github Source Code

Vision
